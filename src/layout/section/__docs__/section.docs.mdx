# Section

A simple container to divide your page into **sections**.

<Playground>
  <Container>
    <Title>Section</Title>
    <Title as="h2" subtitle>
      A simple container to divide your page into <strong>sections</strong>,
      like the one you are currently reading
    </Title>
  </Container>
</Playground>

### Sizing

Use the `size` prop to resize the `<Section>`.

<Playground>
  {() => {
    const sizes = SECTION_DEFAULTS.sizes.map(size => ({ name: size, size }));
    sizes.splice(0, 0, { name: "normal", size: undefined });

    return sizes.map(({size, name}, i) => (
      <OptionBlock props={{ size }} index={i} key={i}>
        <Section backgroundColor="primary" size={size}>
          <Title textColor="light">Section</Title>
          <Title textColor="light" as="h2" subtitle>
            A {name} container to divide your page into sections,
            like the one you are currently reading
          </Title>
        </Section>
      </OptionBlock>
    ));

}}

</Playground>
