# Table

The inevitable HTML **table**, with special case cells.

<Content>
  <p>
    Use the single <code>{"<Table>"}</code> with the following structure:
  </p>
  <ul>
    <li>
      <code>{"<Table>"}</code> the main <strong>container</strong>
      <ul>
        <li>
          <code>{"<Table.Head>"}</code> the optional <strong>top</strong> part
          of the <code>{"<Table>"}</code>
        </li>
        <li>
          <code>{"<Table.Foot>"}</code> the optional <strong>bottom</strong>{" "}
          part of the <code>{"<Table>"}</code>
        </li>
        <li>
          <code>{"<Table.Body>"}</code> the main <strong>content</strong> of the{" "}
          <code>{"<Table>"}</code>
          <ul>
            <li>
              <code>{"<Table.Row>"}</code> each <code>{"<Table>"}</code>{" "}
              <strong>row</strong>
              <ul>
                <li>
                  <code>{"<Table.Heading>"}</code> a <code>{"<Table>"}</code>{" "}
                  cell <strong>heading</strong>
                </li>
                <li>
                  <code>{"<Table.Cell>"}</code> a <code>{"<Table>"}</code>{" "}
                  <strong>cell</strong>
                </li>
              </ul>
            </li>
          </ul>
        </li>
      </ul>
    </li>
  </ul>
  <p>
    You can set a <code>{"<Table.Row>"}</code> as <strong>selected</strong> by
    using the <code>selected</code> prop.
  </p>
</Content>

  </Table>
</Playground>

### Stripes

Use the `striped` prop on `<Table>` to add stripes to alternating rows to make reading and scanning easier.

<Playground>
  <Table striped>
    <Table.Head>
      <Table.Row>
        <Table.Heading>One</Table.Heading>
        <Table.Heading>Two</Table.Heading>
      </Table.Row>
    </Table.Head>
    <Table.Body>
      {[
        ["Three", "Four"],
        ["Five", "Six"],
        ["Seven", "Eight"],
        ["Nine", "Ten"],
        ["Eleven", "Twelve"]
      ].map(([v1, v2], i) => (
        <Table.Row key={i}>
          <Table.Cell>{v1}</Table.Cell>
          <Table.Cell>{v2}</Table.Cell>
        </Table.Row>
      ))}
    </Table.Body>
  </Table>
</Playground>
