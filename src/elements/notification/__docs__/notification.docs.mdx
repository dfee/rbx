# Notification

Bold **notification** blocks, to alert your users of something.

<Playground>
  <Notification>
    <Delete as="button" />
    Lorem ipsum dolor sit amet, consectetur adipiscing elit lorem ipsum dolor.
    <strong> Pellentesque risus mi</strong>, tempus quis placerat ut, porta nec
    nulla. Vestibulum rhoncus ac ex sit amet fringilla. Nullam gravida purus
    diam, et dictum <a>felis venenatis</a> efficitur. Sit amet, consectetur
    adipiscing elit
  </Notification>
</Playground>

### Colors

Use the `color` prop to specfiy the color of the `<Notification>`.

<Playground>
  {() =>
    DEFAULTS.colors.map((color, i) => (
      <OptionBlock props={{ color }} index={i} key={i}>
        <Notification color={color}>
          <Delete as="button" />
          Primar lorem ipsum dolor sit amet, consectetur adipiscing elit lorem ipsum
          dolor. <strong>Pellentesque risus mi</strong>, tempus quis placerat ut,
          porta nec nulla. Vestibulum rhoncus ac ex sit amet fringilla. Nullam gravida
          purus diam, et dictum <a href="#felis">felis venenatis</a> efficitur. Sit
          amet, consectetur adipiscing elit
        </Notification>
      </OptionBlock>
    ))
  }
</Playground>
