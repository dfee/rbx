# Notification

Bold **notification** blocks, to alert your users of something.

Use the `color` prop to specfiy the color of the `<Notification>`.

### Notify

Use the notify helper to display toast notification

#### Simple string, default options
<Playground><Button color="info" onClick={() => notify("Hello World")}>Notify</Button></Playground>

#### ReactChild, default options
<Playground><Button color="info" onClick={() => notify(<p>Hello <strong>World</strong></p>)}>Notify</Button></Playground>

#### Simple string, default options, centered
<Playground><Button color="info" onClick={() => notify("Hello World", "center")}>Notify</Button></Playground>

#### Custom options, centered
<Playground><Button color="danger" onClick={() => notify({ 
  "children": <p>Hello <strong>World</strong></p>,
  "color": "danger",
  "close": false,
}, "center")}>Notify</Button></Playground>
