# Pagination

A responsive, usable, and flexible **pagination**.

<Content>
  <p>
    The <code>{"<Pagination>"}</code> component contains several other
    components:
  </p>
  <ul>
    <li>
      <code>{"<Pagination.Step>"}</code> for incremental navigation
    </li>
    <li>
      <code>{"<Pagination.List>"}</code> which displays page items:
      <ul>
        <li>
          <code>{"<Pagination.Link>"}</code> for the page numbers
        </li>
        <li>
          <code>{"<Pagination.Ellipsis>"}</code> for range separators
        </li>
      </ul>
    </li>
  </ul>
  <p>
    All components are optional, so you can compose your{" "}
    <code>{"<Pagination>"}</code> as you wish.
  </p>
</Content>

Use the `anchor` standard `disabled` prop on `<Pagination.Step>` to disable it if it's not active.

Use the `current` prop on `<Pagination.Link>` to set it as currently selected.

<Playground>

      <Pagination.Link current>1</Pagination.Link>
      <Pagination.Link>2</Pagination.Link>
      <Pagination.Link>3</Pagination.Link>
    </Pagination.List>
  </Pagination>
</Playground>

### Alignments

Use the `align` prop on `<Pagination>` to force the alignment of the pagination items: `<Pagination.List>` and `<Pagination.Step>`.

By default on **touch** devices `<1024px`, the `<Pagination.List>` is located on the _left_ and the `<Pagination.Step>`s are located on the `right`.

<Playground>
  {() => {
    const alignments = PAGINATION_DEFAULTS.alignments.map(align => ({ align, name: align }));
    alignments.splice(0, 0, { align: undefined, name: "left" });

    return alignments.map(({ align, name }, i) => (
      <OptionBlock props={{ align }} index={i} key={i}>
        <Pagination align={align}>
          <Pagination.Step align="previous">Previous</Pagination.Step>
          <Pagination.Step align="next">Next page</Pagination.Step>
          <Pagination.List>
            <Pagination.Link>1</Pagination.Link>
            <Pagination.Ellipsis />
            <Pagination.Link>45</Pagination.Link>
            <Pagination.Link current>46</Pagination.Link>
            <Pagination.Link>47</Pagination.Link>
            <Pagination.Ellipsis />
            <Pagination.Link>86</Pagination.Link>
          </Pagination.List>
        </Pagination>
      </OptionBlock>
    ));

}}

</Playground>

### Rounded

Use the `rounded` prop on `<Pagination>` to have rounded `<Pagination.Link>` and `<Pagination.Step>` items.

### Sizes

Use the `size` prop on `<Pagination>` to change the size.

<Playground>
 {() => {
    const sizes = PAGINATION_DEFAULTS.sizes.map(size => ({ size, name: size }));
    sizes.splice(1, 0, { size: undefined, name: "normal" });

    return sizes.map(({ size, name }, i) => (
      <OptionBlock props={{ size }} index={i} key={i}>
        <Pagination size={size}>
          <Pagination.Step align="previous">Previous</Pagination.Step>
          <Pagination.Step align="next">Next page</Pagination.Step>
          <Pagination.List>
            <Pagination.Link>1</Pagination.Link>
            <Pagination.Ellipsis />
            <Pagination.Link>45</Pagination.Link>
            <Pagination.Link current>46</Pagination.Link>
            <Pagination.Link>47</Pagination.Link>
            <Pagination.Ellipsis />
            <Pagination.Link>86</Pagination.Link>
          </Pagination.List>
        </Pagination>
      </OptionBlock>
    ));

}}

</Playground>
