# Tab

Simple responsive horizontal navigation **tabs**, with different styles.

The default tabs style has a single border at the bottom.

<Playground>
  <Tab.Group>
    <Tab active>Pictures</Tab>
    <Tab>Music</Tab>
    <Tab>Videos</Tab>
    <Tab>Documents</Tab>
  </Tab.Group>
</Playground>

### Alignments

Use the `align` prop of `<Tab.Group>` to align the tabs list.

<Playground>
  {() => {
    const alignments = TAB_GROUP_DEFAULTS.alignments.map(align => ({ align, name: align }));
    alignments.splice(0, 0, { align: undefined, name: "left" });

    return alignments.map(({ align, name }, i) => (
      <OptionBlock props={{ align }} index={i} key={i}>
        <Tab.Group align={align}>
          <Tab active>Pictures</Tab>
          <Tab>Music</Tab>
          <Tab>Videos</Tab>
          <Tab>Documents</Tab>
        </Tab.Group>
      </OptionBlock>
    ))

}}

</Playground>

### Sizes

Use the `size` prop of `<Tab.Group>` to set the size.

<Playground>
  {() => {
    const sizes = TAB_GROUP_DEFAULTS.sizes.map(size => ({ name: size, size }));
    sizes.splice(1, 0, { name: "normal", size: undefined });

    return sizes.map(({ size, name }, i) => (
      <OptionBlock props={{ size }} index={i} key={i}>
        <Tab.Group size={size}>
          <Tab active>Pictures</Tab>
          <Tab>Music</Tab>
          <Tab>Videos</Tab>
          <Tab>Documents</Tab>
        </Tab.Group>
      </OptionBlock>
    ))

}}

</Playground>

### Kinds

Use the `kind` prop on `<Tab.Group>` to change the styling of the tabs.

With the `kind` prop as `"boxed"`, you'll have classic style **borders**.

<Playground>
  <Tab.Group kind="boxed">
    <Tab active>
      <Icon size="small">
        <FontAwesomeIcon icon={faImage} />
      </Icon>
      <span>Pictures</span>
    </Tab>
    <Tab>
      <Icon size="small">
        <FontAwesomeIcon icon={faMusic} />
      </Icon>
      <span>Music</span>
    </Tab>
    <Tab>
      <Icon size="small">
        <FontAwesomeIcon icon={faFilm} />
      </Icon>
      <span>Videos</span>
    </Tab>
    <Tab>
      <Icon size="small">
        <FontAwesomeIcon icon={faFileAlt} />
      </Icon>
      <span>Documents</span>
    </Tab>
  </Tab.Group>

</Playground>

With the `kind` prop as `"toggle"`, you'll have **mutually exclusive** tabs, like radio buttons.

<Playground>
  <Tab.Group kind="toggle">
    <Tab active>
      <Icon size="small">
        <FontAwesomeIcon icon={faImage} />
      </Icon>
      <span>Pictures</span>
    </Tab>
    <Tab>
      <Icon size="small">
        <FontAwesomeIcon icon={faMusic} />
      </Icon>
      <span>Music</span>
    </Tab>
    <Tab>

        <FontAwesomeIcon icon={faFilm} />
      </Icon>
      <span>Videos</span>
    </Tab>
    <Tab>
      <Icon size="small">
        <FontAwesomeIcon icon={faFileAlt} />
      </Icon>
      <span>Documents</span>
    </Tab>
  </Tab.Group>

</Playground>

### Fullwidth

Use the `fullwidth` prop on `<Tab.Group>` to expand the tabs to take up all available horizontal width.

<Playground>
  <Tab.Group fullwidth>
    <Tab>
      <Icon size="small">
        <FontAwesomeIcon icon={faAngleLeft} />
      </Icon>
      <span>Left</span>
    </Tab>
    <Tab>
      <Icon size="small">
        <FontAwesomeIcon icon={faAngleUp} />
      </Icon>
      <span>Up</span>
    </Tab>
    <Tab>
      <span>Right</span>
      <Icon size="small">
        <FontAwesomeIcon icon={faAngleRight} />
      </Icon>
    </Tab>
  </Tab.Group>
</Playground>
