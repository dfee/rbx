# Breadcrumb

A simple **breadcrumb** component to improve your navigation experience.

<Playground>
  <Breadcrumb>
    <Breadcrumb.Item>rbx</Breadcrumb.Item>
    <Breadcrumb.Item>Documentation</Breadcrumb.Item>
    <Breadcrumb.Item>Components</Breadcrumb.Item>
    <Breadcrumb.Item active>Breadcrumb</Breadcrumb.Item>
  </Breadcrumb>
</Playground>

### Alignments

Use the `align` prop to center or align-right the breadcrumbs.
Leaving this prop unspecified, or setting it to `undefined`, causes it to be **left** aligned.

<Playground>
  {() => {
    const alignments = [
      { align: undefined, name: "normal" },
      ...BREADCRUMB_DEFAULTS.alignments.map(align => ({ align, name: align })),
    ];

    return alignments.map(({ align, name }, i) => (
      <OptionBlock props={{ align }} index={i} key={i}>
       <Breadcrumb align={align}>
          <Breadcrumb.Item>rbx</Breadcrumb.Item>
          <Breadcrumb.Item>Documentation</Breadcrumb.Item>
          <Breadcrumb.Item>Components</Breadcrumb.Item>
          <Breadcrumb.Item active>Breadcrumb</Breadcrumb.Item>
        </Breadcrumb>
      </OptionBlock>
    ));

}}

</Playground>

You can use the `<Icon>` component within the `<Breadcrumb.Item>` for additional flavor.

    <Breadcrumb.Item>
      <Icon size="small">

      </Icon>

    <Breadcrumb.Item>

    <Breadcrumb.Item active>

      </Icon>
      Breadcrumb

  </Breadcrumb>
</Playground>

### Separators

Use the `separator` prop of `<Breadcrumb>` to specify an alternative separator.

<Playground>
  {() => {
    const separators = [
      { name: "forward-slash", separator: undefined },
      ...BREADCRUMB_DEFAULTS.separators.map(separator => ({ name: separator, separator }))
    ];

    return separators.map(
      ({name, separator}, i) => (
        <OptionBlock props={{ separator }} index={i} key={i}>
          <Breadcrumb separator={separator}>
            <Breadcrumb.Item>breadcrumb</Breadcrumb.Item>
            <Breadcrumb.Item>separator</Breadcrumb.Item>
            <Breadcrumb.Item active>{name}</Breadcrumb.Item>
          </Breadcrumb>
        </OptionBlock>
      )
    );

}}

</Playground>

### Sizes

Use the `size` prop on `<Breadcrumb>` to specify the size of the breadcrumbs.

<Playground>
  {() => {
    const sizes = BREADCRUMB_DEFAULTS.sizes.map(size => ({ name: size, size }));
    sizes.splice(1, 0, { name: "normal", size: undefined });

    return sizes.map(({ size, name }, i) => (
      <OptionBlock props={{ size }} index={i} key={i}>
        <Breadcrumb size={size}>
          <Breadcrumb.Item>breadcrumb</Breadcrumb.Item>
          <Breadcrumb.Item>size</Breadcrumb.Item>
          <Breadcrumb.Item active>{name}</Breadcrumb.Item>
        </Breadcrumb>
      </OptionBlock>
    ));

}}

</Playground>
